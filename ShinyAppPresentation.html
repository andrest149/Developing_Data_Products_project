Motivation

This shiny application is part of the Developing Data Products course from the Coursera Data Science specialization from John Hopkins University Bloomberg School of Public Health.

My Shiny App

On my shiny app, the user selects four normal distribution function obtained from a set of experimental data. The selection can be compared with other set of data. To compare the values, a target was done with the a slidebar.

My shiny app can be found here: https://andrest149.shinyapps.io/proyecto/

The source code is available at my github repo: https://github.com/andrest149/Developing_Data_Products_project
